# Audit Skill Repo

Description: Audit this repository for skill quality, missing references, drift between source skills and agent-facing examples, and README discoverability gaps.

## Steps

1. Review the changed or targeted skill folders.
2. Check whether `SKILL.md` is overloaded and should delegate to `references/`.
3. Check whether `.agents/skills/`, `.github/`, `.windsurf/`, `AGENTS.md`, and `GEMINI.md` are aligned with the current source skills.
4. Check whether `README.md` still reflects the repo's public-facing examples.
5. Report concrete gaps in priority order.
6. If asked to implement fixes, apply only the necessary edits.
