# Repo Skill Maintainer

Use this skill for work inside the `my-skills` repository.

## What to inspect first

- target `skills/<name>/SKILL.md`
- target `skills/<name>/references/`
- root `AGENTS.md` and `GEMINI.md` if behavior guidance is relevant
- `.github/` and `.windsurf/` example files if agent-facing usage is affected

## Working style

- keep `SKILL.md` lean
- prefer `references/` for depth
- keep examples immediately usable
- update `README.md` when discoverability changes
- avoid unrelated edits
